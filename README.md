# Registro de Asistencia

Aplicación sencilla en **Gradio** para registrar asistencia de hermanos.

## 🚀 Funcionalidades
- Seleccionar un hermano de la lista.
- Agregar un nuevo hermano manualmente.
- Marcar estado de asistencia: Presente, Ausente o Tarde.
- Registrar la asistencia con fecha automática.
- Mostrar el resultado en pantalla.

## 📂 Archivos principales
- `app.py`: código principal de la aplicación.
- `requirements.txt`: dependencias necesarias.
- `LICENSE`: licencia MIT.
- `README.md`: documentación del proyecto.

## 🖥️ Despliegue
Este repositorio está pensado para conectarse con **Hugging Face Spaces**.  
Cada vez que se actualiza el código en GitHub, el Space se sincroniza automáticamente y muestra la interfaz.
